Developer API

Full Control via REST & WebSocket

Every feature is accessible through clean, documented API endpoints.

POST /send_command

Enqueue a prompt to the active AI chat

POST /start-new-chat

Start a fresh chat session

POST /switch_chat

Switch to a specific conversation by title

POST /toggle_auto_run

Toggle the auto-click Run button feature

POST /toggle_auto_allow

Toggle the auto-approve permission setting

GET /stats

Get cumulative usage metrics

GET /get_command

Poll for queued commands and current state

WS ws://localhost:9812

Real-time chat output stream
